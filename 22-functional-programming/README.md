# Functional Programming

This section covers the core concepts of functional programming in Python.

## Topics covered:
- **Lambda functions**: Anonymous one-line functions.
- **Map & Filter**: Tools for functional data processing.
- **Recursion**: Functions calling themselves to solve complex problems.

## Tasks:
1. `01_filter_map_lambda.py` — Filtering a list and transforming data using lambda.
2. `02_recursive_factorial.py` — Calculating factorial using a recursive approach.
3. `03_combined_recursion_filter.py` — A complex task combining filtering and recursion.
