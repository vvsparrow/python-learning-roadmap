# Python Learning Roadmap

My journey from zero to Junior Python Developer.

## Goal
To master Python backend development and find a remote job.

## Principles
- **Manual Coding:** No copy-paste, no autocompletion. Only manual typing to
build muscle memory.
- **English First:** All code comments and documentation are in English.
- **Consistency:** Learning every day.

## Content
- **22-functional-programming:** Map, Filter, Lambda, and Recursion tasks.
