# OOP Basics

This section introduces the fundamental principles of Object-Oriented
Programming (OOP) in Python.

## Pillars of OOP:
- **Encapsulation**: Hiding internal implementation details and protecting data.
- **Inheritance**: Creating new classes based on existing ones to reuse code.
- **Polymorphism**: Defining a common interface for different classes.
